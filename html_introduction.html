Learn HTML: Introduction

HTML (HyperText Markup Language) is the backbone of every webpage on the internet. It's what gives content its structure and meaning — from headings and paragraphs to links, images, and videos.

Whether you're building a personal blog, a corporate site, or a complex web app, your journey starts here — with HTML.

What is HTML?

  • HTML stands for HyperText Markup Language
  • It’s the standard markup language used to create web pages
  • It describes the structure of content using elements and tags
  • It works hand-in-hand with CSS and JavaScript

Where is HTML Used?

  • Websites and blogs
  • Email templates
  • Mobile app interfaces (with frameworks)
  • eCommerce and CMS platforms
  • Online forms and interactive tools

A Simple HTML Document

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
Try it Yourself

What is an HTML Element?

An HTML element is defined by a start tag, content, and an end tag:

<tagname>Content goes here...</tagname>

Example Elements

  • <h1>My First Heading</h1>
  • <p>My first paragraph.</p>
Note: HTML tags are not case-sensitive. However, lowercase is preferred for consistency and readability.

HTML Page Structure

<!DOCTYPE html>
<html>
  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Why Learn HTML?

  • It’s the first step in web development
  • You can build static websites quickly
  • It forms the base for learning CSS, JS, and frameworks
  • In-demand skill for jobs and freelance work

Quick Facts

  • Invented by Tim Berners-Lee in 1991
  • HTML5 is the latest version (released in 2014)
  • Used by over 95% of websites globally

Did You Know?

  • You can embed audio and video directly using HTML5
  • HTML works without needing any special software — just a browser and a text editor
  • Many job interviews for web dev start with HTML basics

HTML History

  • 1991: HTML introduced by Tim Berners-Lee
  • 1995: HTML 2.0 standardized
  • 1997: HTML 4.0 released
  • 2014: HTML5 finalized by W3C
